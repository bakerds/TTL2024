# Routing

## Routing Protocols

**Static Routing**: Static routing works the same as it does in IPv4

**BGP**: MP-BGP is an extension to BGP4 that supports additional address families:

- IPv4 unicast
- IPv4 multicast
- IPv6 unicast
- IPv6 multicast
- MPLS VPN families

**OSPF**: OSPF supports IPv6 in OSPFv3

**EIGRP**: EIGRP supports IPv6
